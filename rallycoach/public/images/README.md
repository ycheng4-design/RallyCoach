# RallyCoach Hero Images

Place your badminton images here:

## Required Images

1. **hero-team-bg.jpg** - Team lineup photo (background layer)
   - Recommended: Wide shot of badminton players standing in a row
   - Used as the backdrop in the 3-layer hero section
   - Will be faded/masked with CSS

2. **hero-jumper.png** - Single player doing a jump smash (foreground layer)
   - MUST be a transparent PNG (no background)
   - This player "pops out" in front of the text for 3D depth
   - Recommended: High-resolution action shot

3. **arena-gear.jpg** - Badminton rackets/equipment close-up
   - Used in the "Arena" gallery section (left panel)

4. **arena-match.jpg** - Badminton match/tournament photo
   - Used in the "Arena" gallery section (center panel, largest)

5. **arena-drills.jpg** - Player training/drilling photo
   - Used in the "Arena" gallery section (right panel)

6. **court-tactics.jpg** - Badminton court overhead or tactical view
   - Used in the "Visualize Victory" feature section

## Image Specs
- hero-team-bg: 1920x1080 or wider, JPG
- hero-jumper: 800x1200+, transparent PNG
- arena images: 800x600+, JPG
- court-tactics: 800x600+, JPG
