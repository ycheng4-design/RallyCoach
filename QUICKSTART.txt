BADMINTON ANALYZER - QUICK START CHECKLIST
================================================================================

Follow these steps in order:

[ ] STEP 1: Get Supabase Credentials
    - Go to https://supabase.com
    - Create new project (wait 2-3 minutes)
    - Settings ‚Üí API ‚Üí Copy:
      * Project URL
      * anon key
      * service_role key

[ ] STEP 2: Get Gemini API Key
    - Go to https://aistudio.google.com/app/apikey
    - Create API Key
    - Copy the key

[ ] STEP 3: Set Up Supabase Database
    - Open Supabase dashboard ‚Üí SQL Editor
    - Copy ALL content from supabase-setup.sql
    - Paste and Run
    - Storage ‚Üí New bucket ‚Üí Name: "videos" ‚Üí Private

[ ] STEP 4: Configure backend/.env
    GEMINI_API_KEY=<paste your gemini key>
    SUPABASE_URL=<paste your supabase url>
    SUPABASE_ANON_KEY=<paste your anon key>
    SUPABASE_SERVICE_KEY=<paste your service key>

[ ] STEP 5: Configure frontend/.env
    VITE_SUPABASE_URL=<paste your supabase url>
    VITE_SUPABASE_ANON_KEY=<paste your anon key>
    VITE_BACKEND_URL=http://localhost:8000

[ ] STEP 6: Install Backend Dependencies
    cd backend
    python -m venv venv
    venv\Scripts\activate
    pip install -r requirements.txt

[ ] STEP 7: Install Frontend Dependencies
    cd frontend
    npm install

[ ] STEP 8: Verify Setup
    python check-setup.py
    (All checks should pass)

[ ] STEP 9: Start Backend
    Double-click: start-backend.bat
    OR manually:
      cd backend
      venv\Scripts\activate
      python main.py

    Should see: "Uvicorn running on http://0.0.0.0:8000"

[ ] STEP 10: Start Frontend (new terminal)
    Double-click: start-frontend.bat
    OR manually:
      cd frontend
      npm run dev

    Should see: "Local: http://localhost:5173/"

[ ] STEP 11: Test the App
    - Open http://localhost:5173
    - Sign up with email/password
    - Upload a video file
    - Wait for analysis
    - View results!

================================================================================
TROUBLESHOOTING
================================================================================

Backend won't start?
  ‚Üí Check venv is activated (should see (venv) in terminal)
  ‚Üí Run: pip install -r requirements.txt
  ‚Üí Check .env file has real API keys (no "your-project")

Frontend won't start?
  ‚Üí Check node_modules exists
  ‚Üí Run: npm install
  ‚Üí Check .env file is configured

"Connection refused"?
  ‚Üí Make sure backend is running first
  ‚Üí Check http://localhost:8000/health in browser

"Supabase error"?
  ‚Üí Verify .env values are correct
  ‚Üí Check SQL was executed successfully
  ‚Üí Verify storage bucket "videos" exists

Analysis fails?
  ‚Üí Check Gemini API key is valid
  ‚Üí Try shorter video first
  ‚Üí Check backend terminal for errors
